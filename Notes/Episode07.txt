useEffect():-
1.If no dpendency array useEffect is called on every render.
 ex:useEffect(()=>{})
2.If Empty depedency array useEffectis call intial render or first page loads.
 ex: useEffect(()=>{},[])
3.If depedency array in useEffect call dependency array is change useEffect is called.
useEffect(()=>{},[btnLogin])


useState():-
useState is doesn't create a outside functional component and if and else statement.
only create inside the functional componnent and top.


Router:-
Navigation controll for single page applications 
the we need istall it 
npm install react-router-dom

creating new pages then we need use Routers
most of we using createBrowserRouter 
createBrowserRouter:- create list of object then we pass path and element

ex: const app = createBrowserRouter([
    {
        path:'/',
        element:<GafoorFoodApp />,
    },
    {
        path:"/about",
        element:<About />,
    },
    {
        path:"/contact",
        element:<Contact />,
    }
])

RouterProvider:-

RouterProvider is provider component then pass the arugument of the routers of the createBrowserRouter as a children
ex: const root = ReactDOM.createRoot(document.getElementById("root"));

root.render(< RouterProvider router={app}/>);

ChildrenRouter:
React-router is given the name is outlet component then replace the outlet the render component of the path.

Link :-
Link is component is link the path.


Routing:-
Routings are two types in web applications
 1. Client Side Routing => Changing pages without reloading the browser.
 2. Server Side Routing => you make network call the page html is coming form server.
